# Vibe Isometric Sprites (Bonus)

A lightweight bonus folder from the original `vibe-isometric-sprites` repo.

What’s kept in this starter pack:
- the prompt history in `prompts/`
- the reusable `fal-ai-image` skill at the repo root:
  - `.claude/skills/fal-ai-image/`
  - `.agents/skills/fal-ai-image/`

What was removed from this pack version:
- large generated assets
- raw experiment outputs
- heavy media files

Why:
this starter pack is meant to stay practical as a lead magnet / jam download, so the bulky experiment artifacts were trimmed out.
